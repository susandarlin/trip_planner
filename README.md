# trip_planner

A simple static trip planning demo using HTML, Tailwind CSS, and JavaScript.

## Files

- `index.html` — the planner UI built with Tailwind and local JS.
- `app.js` — itinerary generation logic and form handling.

## Usage

1. Open `index.html` in a browser.
2. Enter a destination, number of days, budget, and travel style.
3. Click **Generate itinerary** to see a sample daily plan.
